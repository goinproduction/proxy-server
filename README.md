<ol type="1">
    <li>Cho phép Client truy cập website thông qua các method: GET, POST.
    </li>
    <li>Hỗ trợ giao thức HTTP.
    </li>
    <li>

    </li>
    <li>

        truy cập đến các domain có trong danh sách này.
    </li>
</ol>
